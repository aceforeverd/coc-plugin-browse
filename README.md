# plugin-browse

open plugin home page (at GitHub) with plugin name in vimrc

## Install
`:CocInstall plugin-browse`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-plugin-browse-keymap)`

## Lists

`:CocList demo_list`
